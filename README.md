# 📖 Awesome-books-ES6 <a name="about-project"></a>

> This project was given by Microverse to make our Awesome-books project using ES6 syntax and modules and including Luxon files. Everything was refactored from the previous project including the the function for adding and removing book.

**Awesome-books-ES6** is a project book-list for adding and removing books using ES6 syntax.

### Key Features <a name="key-features"></a>

- **Adding book**
- **Removing book**
- **Navigation**

To get a local copy up and running, follow these steps.

